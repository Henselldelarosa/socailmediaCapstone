# Navigating The Reply Page by postId

## Post Session
At the very top of the page you will see the current post that was clicked on.

<img width="557" alt="Screenshot 2023-03-29 210836" src="https://user-images.githubusercontent.com/97318548/228702627-3b976b2b-bae2-4d0d-a565-abfb561f892d.png">

If the current user owns this post they are able to edit this post by clicking the 'Edit' button at the top that will only show up if the current user owns this post.

<img width="41" alt="Screenshot 2023-03-29 210918" src="https://user-images.githubusercontent.com/97318548/228702921-11c9e983-0036-464f-9fc6-48b77aee6eed.png">

When you click the 'Edit' button a form will show up.

<img width="1055" alt="Screenshot 2023-03-29 210940" src="https://user-images.githubusercontent.com/97318548/228703437-37866063-8d1c-417e-a38b-e55d2e68e4c5.png">

Here you will see the current user info, a cancel button and the current post information. You may enter what you would like to change the post to and also add a image url to the post to be able to display an image with your post.

If you want to cancel any update to this post click the cance button displayed next to the user info and it will take you back to the current post page. 

<img width="76" alt="Screenshot 2023-03-29 211029" src="https://user-images.githubusercontent.com/97318548/228703857-31110aac-f71b-424a-8c55-87d67f52f57e.png">

## Repy Session
Under the post session of the 'post/:postId' page you will see a reply bar where you are able to reply to this post by entering a text on the first bar and optional a image url. This follows the same error handle as the post bar in the home page.

<img width="1114" alt="Screenshot 2023-03-29 211050" src="https://user-images.githubusercontent.com/97318548/228704495-d5bc7014-3676-4033-8b9b-c556865a2659.png">

Under the reply bar you will see all the replies that have been added to the to the current post order from newest to oldest.

<img width="302" alt="Screenshot 2023-03-29 213045" src="https://user-images.githubusercontent.com/97318548/228704756-5a48cc21-67b0-43f6-9879-87a1d08b78a1.png">

By clicking the '...' button a drop manu will appear that will let the owner of the reply edit or delete the reply.

<img width="595" alt="Screenshot 2023-03-29 211228" src="https://user-images.githubusercontent.com/97318548/228705007-e498b422-213c-493a-84cd-9bd8a85035be.png">

By clicking the 'Edit' button you form will appear with the current reply owner info with a cancel button next to it. Under the user info the current reply info will show in which the user may enter what they would like to update the reply to and also add a optional image url. This follows the same error handles as the create post bar. To cancel any updates just click the 'Cancel' button and it will hide the form.
